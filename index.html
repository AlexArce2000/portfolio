Mis Proyectos

CMS

Este proyecto es un Content Management System (CMS) desarrollado con Django y Bootstrap como parte de la materia Ingeniera de Software. El sistema permite gestionar contenidos de manera eficiente, proporcionando funcionalidades de administración, visualización y edición de datos a través de una interfaz amigable.

BookBase

Aplicación web que permite a los usuarios gestionar su colección de libros con facilidad. Desarrollada con Angular en el frontend y Spring Boot en el backend, ofrece todas las operaciones CRUD esenciales: crear, leer, actualizar y eliminar.

Reloj

Este proyecto es un reloj que simula a una estructura del lenguaje C está creado con HTML, CSS y JavaScript. Su propósito es mostrar la hora actual en tiempo real, actualizándose cada segundo.
